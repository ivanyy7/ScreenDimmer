# ScreenDimmer — план

## Цель

Небольшое приложение для Windows: полное затемнение всех мониторов сплошным чёрным (оверлеи), переключение горячей клавишей, иконка в трее, один экземпляр процесса, без прав администратора.

## Маршрут (MVP)

1. Проект WPF на .NET 8, user-mode, манифест `asInvoker`.
2. Низкоуровневый хук клавиатуры (`WH_KEYBOARD_LL`) для выбранной клавиши (сейчас F8).
3. Окна-оверлеи на каждый монитор (`Screen.AllScreens`), позиционирование через Win32 при необходимости.
4. `NotifyIcon`: переключение и выход.
5. Single instance через именованный `Mutex`.
