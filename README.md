# ScreenDimmer

Утилита для Windows: затемнение всех мониторов чёрным оверлеем, переключение по **F8**, иконка в трее, один процесс без прав администратора.

## Сборка

Нужен **.NET 8 SDK**. В каталоге репозитория:

```powershell
dotnet build -c Release
```

Готовый exe: `bin\Release\net8.0-windows\ScreenDimmer.exe`

## Поведение (этап 2)

| Действие | Результат |
|----------|-----------|
| **ЛКМ** по иконке в трее | Показать или скрыть окно «о программе» (тема тёмная/светлая, подсказка по F8) |
| **ПКМ** по иконке | Меню: затемнение (F8), автозапуск, **Exit** |
| **F8** | Включить / выключить затемнение всех мониторов |
| Кнопка «—» в окне | Свернуть окно в трей (приложение работает дальше) |
| Закрытие окна (Alt+F4 и т.п.) | Окно скрывается, процесс **не** завершается |
| **Exit** в меню трея | Выход из приложения |
| Второй запуск exe | Второй экземпляр не стартует (один процесс) |

Окно **не** показывает кнопку в панели задач (`ShowInTaskbar = false`). Настройки темы и размера/положения окна сохраняются в `%LocalAppData%\ScreenDimmer\settings.json`.
